# toanpho24.github.io

My personal portfolio website — built with plain HTML/CSS/JS, hosted on GitHub Pages.

**Live:** [toanpro24.github.io](https://toanpro24.github.io)

## What's Inside

- Single-page portfolio with dark theme and Arsenal-inspired red accent
- Scroll reveal animations via Intersection Observer
- Responsive design (mobile, tablet, desktop)
- Custom SVG favicon
- SEO setup (robots.txt, sitemap.xml, Open Graph meta tags)
- Custom 404 page

## Structure

```
index.html        — Main portfolio page
404.html          — Custom 404 error page
robots.txt        — Search engine crawling rules
sitemap.xml       — Sitemap for indexing
assets/
  favicon.svg     — Site favicon
  hollywood.jpg   — Profile photo
  adobe.jpg       — About section photo
```

## Tech

No build tools, no frameworks, no dependencies. Just HTML, CSS, and vanilla JS.
