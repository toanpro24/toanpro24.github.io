# toanpro24.github.io

My personal portfolio website — built with plain HTML/CSS/JS, hosted on GitHub Pages.

**Live:** [toanpro24.github.io](https://toanpro24.github.io)

## What's Inside

- Single-page portfolio with dark theme and Arsenal-inspired red accent
- Dark/light theme toggle with localStorage persistence
- Page loader animation, scroll progress bar, and cursor glow effect
- Staggered hero entrance and animated stat counters
- Scroll reveal animations via Intersection Observer
- "Currently" status widget (watching/building)
- Resume download button
- Responsive design (mobile, tablet, desktop)
- Custom SVG favicon
- SEO setup (robots.txt, sitemap.xml, Open Graph meta tags)
- Custom 404 page
- Konami code easter egg (open console for a hint)

## Structure

```
index.html        — Main portfolio page
404.html          — Custom 404 error page
robots.txt        — Search engine crawling rules
sitemap.xml       — Sitemap for indexing
assets/
  favicon.svg     — Site favicon
  hollywood.jpg   — Profile photo (hero)
  adobe.jpg       — About section photo
  image.png       — K-Drama project screenshot
  Toan_Pham_Resume.pdf — Downloadable resume
```

## Tech

No build tools, no frameworks, no dependencies. Just HTML, CSS, and vanilla JS.
